<h2>GameMerchStore</h2>
<h4>In my first semester at university, I created a Game Merch Store from scratch as part of my coursework.</h4> 
<h4>This online store is equipped with all the necessary features, such as a sign-in and log-in page, a homepage, and a dedicated selling page showcasing Phasmophobia and Valorant merchandise. </h4>
<h4>During the presentation, the pages containing details about the two chosen games (Phasmophobia and Valorant) were marked as unreleased pages, intended for future launches. 
However, I will make sure to create and provide links to these pages in their respective repositories, so you can easily find them.</h4>
<ul>
  <li>My teammate designed the advertising page and merch for Valorant. You can check out her work <a href="https://github.com/absolutenobrainer333/WEB">here</a>.</li>
  <li>As for the rest concerning the Phasmophobia page and merch (in this very repository) , you can find them in my profile.</li>
</ul>
